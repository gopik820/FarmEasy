## About The Project

![Product Name Screen Shot][product-screenshot]

It is an e-commerce marketplace for buying farming goods and resources. It also gives farmers the exposure they need in order to thrive well in markets and keep up with other sellers in the same business.

### Built With

These are the frameworks that we used:

* Flask
* HTML,CSS
* JavaScript

## Installation

- Clone the repo
 - `https://github.com/gopik820/FarmEasy`
- Install the necessary packages
 - `pip install -r requirements.txt`
- Run the application
 - `python app.py`

The application provides three features:
### Vendor List
List of vendors from whom the products can be purchased.
### New Vendors
Interested vendors can be added.
### Markets Nearby
Explore all the farmer's markets near and around you.
